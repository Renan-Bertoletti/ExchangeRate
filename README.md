# ExchangeRate

Este é um projeto simples em Java, executado via console, que permite converter valores entre diferentes moedas utilizando taxas atualizadas em tempo real por meio da ExchangeRate-API. O programa apresenta um menu com várias opções de conversão e continua sendo executado até que o usuário escolha a opção 7 (Sair).
Deve fazer o cadastro em https://www.exchangerate-api.com/ para receber uma chave e utilizar em ExchangeRate.java para fazer uso do programa

Menu interativo exibido no console Suporte a 6 conversões de moedas: Dólar ⇒ Peso Argentino Peso Argentino ⇒ Dólar Dólar ⇒ Real Real ⇒ Dólar Euro ⇒ Real Real ⇒ Euro Uso da ExchangeRate-API para obter taxas de câmbio atualizadas Desserialização JSON com a biblioteca Gson
